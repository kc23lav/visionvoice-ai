# 🎙️ VisionVoice AI

### One Voice. One Assistant.

VisionVoice AI is an AI-powered accessibility assistant designed to empower visually impaired individuals through voice-first interaction and intelligent assistance.

The platform combines document reading, scene understanding, medicine identification, emergency assistance, and voice navigation into a single accessible solution.

---

## 🚀 Problem Statement

Visually impaired individuals often face challenges in:

* Reading printed documents
* Understanding their surroundings
* Identifying medicines safely
* Accessing emergency assistance
* Navigating digital interfaces independently

Most existing solutions focus on a single functionality, requiring users to switch between multiple applications.

VisionVoice AI addresses this challenge by providing a unified AI-powered accessibility platform.

---

## ✨ Key Features

### 📄 Document Reader

* OCR-based text extraction using EasyOCR
* AI-generated simplified summaries
* Text-to-Speech output

### 👁️ Scene Understanding

* AI-powered image analysis
* Environment and object description
* Accessibility-focused explanations

### 💊 Medicine Detector

* Medicine identification from packaging images
* Purpose and usage information
* Important warnings and precautions
* Voice feedback

### 🚨 Emergency SOS

* Quick emergency access interface
* Designed for future GPS integration
* Accessibility-focused emergency support

### 🎤 Voice Navigation

* OpenAI Whisper Speech Recognition
* Voice-controlled page navigation
* Hands-free interaction

### 🔊 Voice Feedback

* AI responses are automatically spoken aloud
* Improved accessibility and usability
